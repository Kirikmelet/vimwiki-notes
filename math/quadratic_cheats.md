- *Quadratic Formula*

x = (-b +- sqrt(b^2^ - 4ac))/2a

EX.

3x^2^ -5x - 8

a = 3, b = -5, c = -8

x = (5 +- sqrt(-5^2^ - 4(3)(8)))/2(3)

x = (5 +- sqrt(25 + 96))/6

x = (5 +- sqrt(121))/6

x = (5 +- 11)/6

x = (5 + 11)/6

x = (5 - 11)/6

x = 16/6

x = -6/6

b^2^ - 4ac
